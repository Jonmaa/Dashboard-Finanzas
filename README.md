# Finanzas Personales - Dashboard

Un dashboard sencillo, bonito y 100 % funcional para controlar tus ingresos y gastos.

**Live Demo** → https://dashboard-finanzas-jmm.vercel.app/

## Qué tiene

- Añadir ingresos y gastos  
- Balance en tiempo real  
- Gráfico de gastos por categoría 
- Evolución mensual 
- Exportar todo a CSV (Excel en español ok) 
- Modo oscuro / claro 
- Responsive (móvil + escritorio)  
- Todo se guarda en el navegador (localStorage)

## Tecnologías

- React + Vite  
- TypeScript  
- Tailwind CSS  
- Recharts  
- Lucide Icons

## Cómo probarlo

```bash
git clone https://github.com/Jonmaa/Dashboard-Finanzas.git
cd finanzas
npm install
npm run dev
```
Abre http://localhost:5173 y listo!
